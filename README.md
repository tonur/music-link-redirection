# Music Link Redirection
Redirect Spotify, YouTube Music and more to your prefered streaming service, powered by Linktree Songlink/Odesli 

Inspired by [Spotify to YT Music Redirect by Arael](https://addons.mozilla.org/en-US/firefox/addon/spotify-to-yt-music-redirect/).

This extension relies soely on the Linktree Odesli API documented at: <https://linktree.notion.site/API-d0ebe08a5e304a55928405eb682f6741>
The API is rate limited to 10 request per minute.
In the case that this stops working, this extension will too.
